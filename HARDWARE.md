# 硬件范围与接线边界

本文件从公开固件的引脚常量和原始说明整理而来，是**源码推导接口说明**，不是实物原理图、PCB、BOM 采购保证、制造文件、经电气审核的设计或当前公开 commit 的实物接线证据。

## 公开代码涉及的接口

| 接口 | 固件引脚/地址 | 公开默认行为 | 必须另行确认 |
| :-- | :-- | :-- | :-- |
| DHT11 数据 | GPIO4 | 初始化并周期读取 | 模块电压、上拉、线长、失效路径与读数 |
| I²C SDA / SCL | GPIO8 / GPIO9 | 初始化 BH1750、ACD10、BMP280、OLED 的读取/显示路径 | 实物模块、电平、地址、上拉、总线冲突与供电 |
| BH1750 | 常见 `0x23` | 读取光照入口 | 模块地址、标定与可用性 |
| ACD10 | 常见 `0x2A` | 预热与读取入口 | 型号、预热、准确性、CO₂ 语义与电平 |
| BMP280 | 常见 `0x76` | 气压读取入口 | 模块地址、电平和准确性 |
| SSD1306 | 常见 `0x3C` | 本地 OLED 显示 | 屏幕型号、地址、供电与实际显示 |
| 风扇驱动候选 | GPIO5 / GPIO6 | 默认均为输入；opt-in 编译分支固定低电平 | 驱动板、极性、续流、额定电压/电流、公共地与负载 |
| WS2812B 候选 | GPIO38 | 默认输入；opt-in 编译分支显示黑色 | 灯带电源、逻辑电平、限流、共地与真实行为 |
| Wi-Fi/TCP | 无固定公开网络地址 | 默认不连接；可选单向教学遥测 | 可信网络、凭据、地址、隐私与网络安全 |

## 默认输出与电气边界

- `ENABLE_EXPERIMENTAL_ACTUATORS=0` 时，GPIO5、GPIO6、GPIO38 不配置为输出。外部驱动、上拉、供电、接线和负载未知，因此这**不代表**外部负载物理关闭、无误动作或电气安全。
- `ENABLE_EXPERIMENTAL_ACTUATORS=1` 的编译分支只用于验证源码可编译；本公开代码仍将两路风扇信号写为低电平并让 FastLED 输出黑色。它不控制实际风扇/补光，不能据此推断外部模块的有效/无效电平或真实动作。
- ESP32 GPIO 不得直接驱动风扇、灯带或任何高电流/市电负载。真实台架需要合规驱动、限流/续流、独立低压供电和保护；第一次接线/改线/烧录前先断电。
- 当前仓未提供并未核验真实原理图、PCB、Gerber、实物接线、模块型号、供电、保护、温升、EMC、作物环境效果或量产能力。

完整器件边界见 [hardware/BOM.csv](hardware/BOM.csv) 与 [hardware/wiring.md](hardware/wiring.md)。
