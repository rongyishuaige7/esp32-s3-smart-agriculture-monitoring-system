# 来源、净化与公开范围

## 来源边界

- **原始来源：** 桌面中的原始“SmartAgriculture”工程（本机路径不随公开仓发布）。
- **公开候选：** 本仓根目录。它是唯一可写的公开整理副本；新增安全默认、文档、BOM、边界图和门禁均只在此完成。
- **并行旧固件：** 桌面中的另一套 Arduino 风格 `esp32_firmware` 未被自动合并、未复制、未上传；它不是本仓权威固件来源。
- 原工程保持只读：本轮不在原目录初始化 Git、清理、删除、移动、提交、推送或反向覆盖。原 `SmartAgriculture_src_new.zip` 是工作树快照，不是第二权威版本，也不公开。

## 安全来源 allowlist

[`source-allowlist.txt`](source-allowlist.txt) 是对原始工程中保留公开内容或直接来源谱系所需的相对文件清单。它不是原始工程全量文件清单，也不列出已排除的数据层、SQL、账号/登录、历史/阈值/控制页面或用户模型。`scripts/source_manifest.py` **只读取该清单**，不递归扫描原工程，并拒绝原 `App.config`、ZIP、IDE 状态、缓存、构建物、未纳入 `.sln` 的 WinForms 项目和本地配置路径。

本次公开来源范围收紧后，allowlist 中的原始文件按 UTF-8 相对路径、NUL 分隔符、原文件字节和 NUL 分隔符的排序序列计算 SHA-256：

```text
files=26
sha256=7dbbb7c32641ee70f7db43baa759d7ea489168a368e45d02d507205a2e2023ac
```

该摘要只证明当前公开来源清单所列的原始文件集合；它不证明公开候选、许可证、构建、真机、历史版本或原工程全量安全。候选中新增/改写的公开默认代码、文档、BOM、SVG、脚本和许可证不在原始 allowlist 中。

## 可审计公开整理

在不修改原始来源的前提下，候选：

1. 只从 allowlist 单向复制 `.sln`、Avalonia/.NET 8 源码、PlatformIO 固件和可许可的 ACD10 本地库；排除原 `App.config`、ZIP、`bin/`、`obj/`、`.idea/`、`.vscode/` 和 WinForms 项目；
2. 将公开桌面端收口为内置示例数据（非真机）的教学展示：没有登录、账号、数据库、TCP listener、设备控制页、下行命令或阈值同步；
3. 将固件收口为传感器/OLED 观察默认：没有 Wi-Fi/TCP，GPIO5/GPIO6/GPIO38 不配置为输出；显式 opt-in 编译分支分别只验证单向遥测和已知输出保持关闭的源码路径；
4. 不公开实物照片、视频、原理图、PCB、EDA、Gerber、制造文件、真实运行日志、网络配置、环境数据或数据库导出；
5. 补充 README、硬件/安全/协议/状态/验证文档、BOM、源码推导边界图、第三方声明、源扫描、结构门禁、源码契约与 CI。
