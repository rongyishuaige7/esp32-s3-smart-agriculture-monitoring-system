# 源码推导接线边界

下表只列公开固件中的接口常量，不能替代真实原理图、接线、PCB、模块数据手册或电气评审。接线前请按实际模块核对电压、电平和供电。

| 公开源码接口 | ESP32-S3 引脚/地址 | 使用前必须确认 |
| :-- | :-- | :-- |
| DHT11 DATA | GPIO4 | 模块供电、上拉、读数失败和线长 |
| I²C SDA / SCL | GPIO8 / GPIO9 | 所有模块电平、地址、上拉、总线长度和共地 |
| BH1750 / ACD10 / BMP280 / OLED | `0x23` / `0x2A` / `0x76` / `0x3C` 常见地址 | 实际地址、供电、型号和地址冲突 |
| 风扇驱动候选 | GPIO5 / GPIO6 | 合规驱动、极性、续流、电流、独立供电和保护 |
| WS2812B 候选 | GPIO38 | 逻辑电平、独立电源、限流和共地 |

默认构建把 GPIO5、GPIO6、GPIO38 配置为输入。即使 local opt-in 编译通过，外部驱动和负载仍可能受电气设计影响；请勿把“默认不输出”写成“实体负载已关闭”或“电气安全”。
